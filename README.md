## Troubleshooting

- If the launcher does not open, make sure you are on Windows and that the files were fully extracted.
- If a profile does not load the way you expect, confirm that the selected loader matches the version you installed.
- If mod or texture pack searches fail, try again after checking your network connection.
- If the launcher seems to reuse older settings, clear or reset local launcher data and start over.
- If permissions prevent file changes, move the launcher to a folder where your user account can write files.
- If a launch ends early, verify that the selected version, mod loader, and installed content are compatible with one another.
